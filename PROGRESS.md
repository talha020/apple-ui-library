## 📝 Component Guidelines

All components follow:
- Apple HIG design principles
- Shopify-admin aesthetic (borders create depth, not shadows)
- Design tokens from AppleHIGDashboard.jsx
- Responsive, accessible markup
- Clean, semantic HTML
- Inline styles matching token system
- Satoshi font family (with SF Pro Display fallback)
- Consistent color palette (#1D1D1F, #6D6D72, #2C6ECB, #1A8245, #7C3AED, etc.)
- Border radius: 10-12px
- Compact, efficient code structure
- Anti-aliased text rendering
- Transition: 0.2s ease on all interactive elements
