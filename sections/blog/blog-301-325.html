306 — Blog with Code Snippets
TUTORIAL

Custom hooks in React

Learn how to extract and reuse stateful logic with custom hooks.

useDebounce.ts Copy 
function useDebounce(value, delay) {
  const [debounced, setDebounced] = useState(value);
  useEffect(() => {
    const timer = setTimeout(() => setDebounced(value), delay);
    return () => clearTimeout(timer);
  }, [value, delay]);
  return debounced;
}

This hook delays updating the state until the user stops changing the input value.
