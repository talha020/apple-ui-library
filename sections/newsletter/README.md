# Newsletter Components - Apple HIG Style Library

## Overview
This directory contains 500 newsletter signup components (001-500) designed following Apple Human Interface Guidelines with Shopify-admin aesthetics.

## Files Created
- **Total Files**: 20 HTML files
- **Components per File**: 25 components
- **Total Components**: 500 newsletter sections
- **File Range**: newsletter-001-025.html through newsletter-476-500.html
