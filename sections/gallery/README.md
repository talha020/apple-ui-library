# Gallery Section Components

## Overview
Complete collection of 500 gallery components (001-500) following Apple HIG design principles with Shopify admin aesthetics.

## Files Created
- **gallery-001-025.html** through **gallery-476-500.html** (20 files total)
- Each file contains 25 unique gallery components
- Total: 500 gallery section variations

## Design Specifications

### Typography
- **Font Family**: Satoshi, -apple-system, BlinkMacSystemFont
- **Headings**: 600-700 weight
- **Body Text**: 400-500 weight

### Colors
- **Primary Blue**: #2C6ECB
- **Success Green**: #1A8245  
- **Purple**: #7C3AED
- **Backgrounds**: #F6F6F7 (page), #FFFFFF (surface), #F5F5F7 (alt)
- **Text**: #1D1D1F (primary), #6D6D72 (secondary)
- **Borders**: #E3E3E8

### Layout
- **Border Radius**: 10-12px for cards/containers, 8px for buttons
- **Grid Gaps**: 16-32px depending on density
- **Padding**: 60px standard container padding
- **Max Width**: 1440px

## Component Types

### Grid Galleries
- 2-column, 3-column, 4-column, 5-column grids
- Square (1:1), landscape (16:9, 4:3), portrait (3:4) aspect ratios
- Compact and spacious layouts

### Masonry Galleries  
- Pinterest-style column layouts
- Asymmetric grid patterns
- Variable height items

### Carousel/Slider Galleries
- Horizontal scroll galleries
- Thumbnail navigation
- Progress indicators and dots
- Panoramic wide-format sliders

### Portfolio Galleries
- Category filters and tags
- Client information cards
- Project metadata
- Case study layouts

### Interactive Galleries
- Hover overlays and zoom effects
- Lightbox previews
- Quick action buttons
- Social sharing options

### Specialized Formats
- Timeline galleries with dates
- Before/after sliders
- Video gallery grids
- 360-degree previews
- Instagram-style feeds
- Product showcase grids

## Usage
Open any HTML file in a browser to view the 25 gallery components in that range. All components use inline styles for easy copying and customization.

## File Structure
Each HTML file follows this pattern:
```html
<!DOCTYPE html>
<html lang="en">
<head>
  <!-- Meta tags and Satoshi font -->
  <style>/* Shared base styles */</style>
</head>
<body>
  <!-- 25 gallery sections with labels -->
</body>
</html>
```

## Notes
- All components are self-contained with inline styles
- Responsive grid layouts using CSS Grid
- Consistent 12px border-radius on containers
- Transitions set to 0.2s ease for interactive elements
- Anti-aliased text rendering enabled
